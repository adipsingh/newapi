Additional Features

We have worked diligently to make sure Attachmore provides all you need to file share effectively.

Secure Links

Uploads and downloads occur using 256bit AES encyrption. Downloads links can be password protected and require downloaders to provide name, email, or company.

Auto Expire - Auto Purge

Increase security by limiting access. Download Links can expire based on days or downloads. Also purge dates can be set to automtically delete older files from storage.

Notifications and Tracking

Receive email or text messages when files are downloaded. All download activity for each link is logged and recorded in your account later review.

Long Term Archiving

Archive all files or just certain files uploaded to Attachmore. Archived files with expired download links can be reactivated to allow recipients to download again.

Guest Links

Create limited use links that others can use to upload files to you. Email or text notifications will let you know files have arrived.

Payment for Download

Need to collect payment for delivery of your digital assets? Attachmore makes it easy. Require payment for download on any of your files at any time.
